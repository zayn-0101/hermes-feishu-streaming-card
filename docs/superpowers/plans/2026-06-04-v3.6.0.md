# V3.6.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship V3.6.0 as an operations-focused release for ordinary Hermes + Feishu users.

**Core Promise:** Users can install, diagnose, safely repair, and operate Hermes Feishu streaming cards without guessing whether hook state, media delivery, bot routing, or Hermes compatibility is broken.

**Out of Scope:** Docker packaging is explicitly excluded for this version. Public release, tag creation, and GitHub release creation require final user confirmation.
